3. Create helper function `visitToBounds(lat: number, lng: number): L.LatLngBoundsExpression`:
   - `const offset = 0.00005;` (~5 meters at Palo Alto latitude)
   - Return `[[lat - offset, lng - offset], [lat + offset, lng + offset]]`

4. Create `FitBounds` inner component:
   - Props: `{ visits: Visit[] }`
   - Uses `useMap()` hook
   - On mount and when visits change, if visits.length > 0: compute `L.latLngBounds` from all visit coordinates, call `map.fitBounds(bounds, { padding: [50, 50] })`
   - If visits.length === 0: set view to Palo Alto default `[37.4419, -122.143]` zoom 13
   - Return `null`

5. Create `VisitRectangle` component (memoized with React.memo):
   - Props: `{ visit: Visit, color: string, isSelected: boolean, onClick: () => void }`
   - Compute bounds via `visitToBounds(visit.latitude, visit.longitude)`
   - Render `<Rectangle>` with:
     - `bounds={bounds}`
     - `pathOptions={{ color: isSelected ? '#ffffff' : color, fillColor: color, fillOpacity: isSelected ? 0.9 : 0.6, weight: isSelected ? 3 : 1 }}`
     - `eventHandlers={{ click: onClick }}`

6. Main `DashboardMap` component:
   - Props: `{ visits, sessionColorMap, selectedVisitId, onSelectVisit }`
   - Render `<MapContainer>` with:
     - `center={[37.4419, -122.143]}` (Palo Alto default)
     - `zoom={13}`
     - `className="w-full h-full"`
     - `style={{ background: '#18181b' }}` (zinc-900 for dark theme before tiles load)
   - Inside MapContainer:
     - `<TileLayer>` with dark tile URL: `https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}{r}.png` and attribution for CartoDB
     - `<FitBounds visits={visits} />`
     - Map over visits: for each visit, render `<VisitRectangle>` with `key={visit.id}`, color from `sessionColorMap.get(visit.session_id) ?? '#3B82F6'`, `isSelected={visit.id === selectedVisitId}`, `onClick={() => onSelectVisit(visit.id)}`

7. Export default `DashboardMap`

Note: This component is already dynamically imported with `ssr: false` in DashboardShell (from Plan 01), so direct Leaflet imports are safe. Use CartoDB dark tiles to match the dark theme of the app.
