    Create `src/lib/route-sort.ts` with a `sortKnocksWalkingOrder` function.

    ```typescript
    export function sortKnocksWalkingOrder<T extends { latitude: number; longitude: number; address: string | null }>(
      knocks: T[]
    ): T[]
    ```

    **Algorithm â€” serpentine street walking:**

    1. **Group by street name:** Parse the address string to extract the street name (everything after the house number). Group knocks by normalized street name (lowercase, trimmed). Knocks with null/unparseable addresses go into a "no street" group sorted by latitude.

    2. **Order streets geographically:** Sort street groups by their average latitude (north to south), so the canvasser works through nearby streets sequentially.

    3. **Serpentine within each street:**
       - Parse the house number from the address (first numeric token).
       - Split into odd numbers and even numbers.
       - Sort odd numbers ascending (walking down one side of the street).
       - Sort even numbers descending (walking back on the other side).
       - Concatenate: odds first, then evens reversed. This creates the "down one side, back the other" pattern.
       - If house number can't be parsed, append to end of that street group.

    4. **Return flat array** of all knocks in walking order.

    **Helper function** to extract street name and house number from address:
    ```typescript
    function parseAddress(address: string): { houseNumber: number | null; street: string } {
      const match = address.match(/^(\d+)\s+(.+)$/);
      if (!match) return { houseNumber: null, street: address };
      return { houseNumber: parseInt(match[1], 10), street: match[2].toLowerCase().trim() };
    }
    ```

    This is a pure function with no side effects â€” easy to test and reason about.
