  <name>Task 1: Drawing state hook and geometry utilities</name>
  <files>src/lib/drawing.ts</files>
  <action>
    Create `src/lib/drawing.ts` with:

    1. **Types:**
       ```typescript
       export type DrawingPoint = { lat: number; lng: number; id: string };
       export type DrawingState = {
         points: DrawingPoint[];
         isDrawing: boolean;
         isClosed: boolean;
         resumeFromId: string | null; // point ID to resume drawing from
       };
       ```

    2. **`useDrawingState` hook** — manages the drawing interaction state:
       - `startDrawing()` — enters drawing mode (isDrawing=true)
       - `stopDrawing()` — exits drawing mode (isDrawing=false), clears resumeFromId
       - `addPoint(lat, lng)` — appends a new point with a generated UUID id. If resumeFromId is set, inserts after that point in the array. Clears resumeFromId after adding.
       - `removePoint(id)` — removes point by id. If only 0-1 points remain, reset isClosed to false.
       - `closePolygon()` — sets isClosed=true (only if 3+ points exist)
       - `resumeFrom(id)` — sets resumeFromId, enters drawing mode
       - `clear()` — resets everything to initial state
       - `getCoordinates()` — returns array of [lat, lng] tuples for all points (the polygon/polyline coords)

    3. **`isPointNearTarget` utility:**
       ```typescript
       export function isPointNearTarget(
         clickLatLng: L.LatLng,
         targetLatLng: L.LatLng,
         map: L.Map,
         pixelThreshold?: number // default 15
       ): boolean
       ```
       Convert both to pixel coordinates via `map.latLngToContainerPoint()`, check if distance < threshold. This handles zoom-independent click detection for closing polygons and selecting points.

    4. **`getPolygonBounds` utility:**
       ```typescript
       export function getPolygonBounds(points: DrawingPoint[]): {
         minLat: number; maxLat: number; minLng: number; maxLng: number;
       } | null
       ```
       Returns bounding box of the drawn shape, or null if < 2 points.

    Use `crypto.randomUUID()` for point IDs (available in all modern browsers).

  <action>
    Create `src/components/dashboard/DrawingMap.tsx` — a map component specifically for the route planning drawing interaction. This is a NEW component (not modifying DashboardMap.tsx). It will be integrated into the dashboard in Plan 03.

    **Component structure:**

    ```typescript
    interface DrawingMapProps {
      onPolygonComplete: (points: DrawingPoint[]) => void;
      onCancel: () => void;
    }
    ```

    **Implementation:**

    1. **Map setup:** Same MapContainer/TileLayer pattern as DashboardMap (PALO_ALTO_CENTER, dark tiles via `map-tiles-dark` class, ResizeObserver pattern for container readiness).

    2. **Drawing mode toggle:** A "Make Selection" button (absolute positioned top-right, z-[1000]) that calls `startDrawing()`. When drawing, button changes to "Cancel" and calls `stopDrawing()` + `onCancel()`.

    3. **Map click handler** using `useMapEvents`:
       - When `isDrawing` and map is clicked:
         - Check if click is near the FIRST point (using isPointNearTarget) — if so, close polygon
         - Check if click is near ANY existing point — if so, remove that point (delete behavior)
         - Otherwise, add a new point at the click location
       - When NOT drawing and a point is clicked → call `resumeFrom(pointId)` to re-enter drawing from that point

    4. **Point markers:** Render each DrawingPoint as a Leaflet CircleMarker (radius 8, white fill, blue border). First point gets a slightly larger radius (10) and green fill when 3+ points exist (visual "close here" affordance).

    5. **Connecting lines:** Render a Polyline through all points using react-leaflet `<Polyline>`. If polygon is closed, connect last point back to first. Use dashed blue line style (weight 2, color '#3B82F6', dashArray '8 4').

    6. **Live preview line:** A separate Polyline from the last placed point to the current cursor position. Track cursor with `useMapEvents({ mousemove })`. Use lighter color and thinner weight (weight 1, color '#60A5FA', dashArray '4 4'). Only show when `isDrawing` and points.length > 0.

    7. **Escape key handling:** Add `useEffect` with `keydown` listener. When Escape pressed and isDrawing, call `stopDrawing()`.

    8. **"Create Run Route" button:** Only visible when polygon is closed (isClosed=true). Positioned bottom-center. Calls `onPolygonComplete(drawing.points)`. Style: bg-green-600 text-white px-6 py-3 rounded-lg shadow-lg.

    9. **Clear button:** Small "Clear" link next to "Create Run Route" that calls `drawing.clear()`.

    Use react-leaflet components (CircleMarker, Polyline, useMapEvents) — no external drawing libraries needed. The interaction is simple enough for a custom implementation per CONTEXT.md's "Claude's Discretion" on library choice.
